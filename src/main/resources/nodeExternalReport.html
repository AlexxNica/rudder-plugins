External Reports for Node plugin configuration

This plugin allows to add external reports on nodes details. You can configure directories where the node hostname or uuid will be look for. If it is found, a link is created in a new tab in node details, allowing to display the report content.

  • File used to configure the reports: [report confog file]
  • Name of the tab in node detail: [tab title]


Report with key "[key]"

  • Title: [title]
  • Description: [description]
  • Root directory: [root directory]
  • Example of file name for node "my-node.local": [file name example]
